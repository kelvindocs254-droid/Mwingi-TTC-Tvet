# Mwingi TTC App v1.0.0

## 🎉 Initial Release

The official mobile app for Mwingi Teachers Training College is now available!

## ✨ Features

### 📱 User Interface
- **Home Screen** - Featured courses and latest announcements
- **Courses** - Complete programs offered (DPTE, DECTE, Upgrade programs)
- **Admission** - Application guide and form download
- **About Us** - College mission, vision, history, and core values
- **Contact** - College location, phone, email, and postal address

### 🔧 Technical Features
- Built with **Jetpack Compose** for modern Android UI
- **Material Design 3** for beautiful, accessible interface
- **Navigation Compose** for smooth screen transitions
- **Firebase Integration** for backend services
- **Gemini API Ready** for AI-powered features
- **Room Database** for local data persistence
- **Retrofit & OkHttp** for reliable networking
- **Moshi** for JSON serialization

## 📦 What's Included

- ✅ 5 main navigation screens

## 📋 Requirements

- **Android**: 7.0 (API 24) or higher
- **RAM**: Minimum 2GB
- **Storage**: ~15MB for app installation
- **Internet**: Required for full functionality

## 🚀 Installation

### Method 1: Download from Release
1. Download `app-release.apk` from this release
2. Transfer to your Android device
3. Go to **Settings → Security → Unknown Sources** and enable it
4. Tap the APK file to install
5. Launch the app!

### Method 2: Using ADB (Android Debug Bridge)
```bash
adb install app-release.apk
adb shell am start -n com.example/com.example.MainActivity
```
